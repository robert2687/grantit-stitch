# RMD26 Multi-Agent Grant Intelligence System - PRD ## 1. Overview RMD26 is an autonomous, global grant-management platform designed to identify, evaluate, and apply for funding opportunities in the AI, ML, Cloud, and Digital Transformation sectors. The system utilizes a multi-agent architecture (Search, Evaluation, Copywriter, Admin, and Orchestrator) to streamline the entire grant lifecycle. ## 2. Core Agents & Functionality ### 2.1 Grant-Search Agent (Autonomous Scanner) * **Mission:** Continuous global scanning for grants (Government, Horizon Europe, Private Foundations). * **Data Points:** Program name, Region, Amount, Deadline, Eligibility, Fit Score (0-100%). * **Outputs:** Daily summaries, weekly strategic reports. ### 2.2 Grant Evaluation Agent (Success Analyst) * **Mission:** Assess RMD26's fit and probability of success. * **Criteria:** Eligibility, Thematic Fit, Innovation Strength, TRL, Geographic Fit, Competition Level. * **Outputs:** Detailed scoring tables, risk assessment, Go/No-Go recommendations. ### 2.3 Grant Proposal Copywriter Agent * **Mission:** Draft high-impact, compliant grant applications. * **Components:** Executive Summary, Problem Statement, Technical Methodology, Work Packages, Budget Justification. * **Style:** Persuasive, evidence-based, structured for evaluators. ### 2.4 Grant Administration Assistant Agent * **Mission:** Manage compliance, documentation, and deadlines. * **Tasks:** Task management, document tracking (checklists), form/template preparation, partner coordination. * **Outputs:** Submission Readiness Reports. ### 2.5 System Orchestrator * **Mission:** Coordinate all agents and present a unified view. * **Flow:** Search -> Filter -> Score -> Go/No-Go -> Draft -> Submit. ## 3. UI/UX Requirements * **Toggle Sidebar:** Users must be able to hide/show the navigation menu to maximize workspace. * **Desktop & Mobile:** Fully responsive web application. * **Design Aesthetic:** Professional, data-rich but clean, utilizing high-contrast elements for clarity in data tables and reports. * **Visual Priority:** AI/ML, Autonomous Agents, Cloud, SaaS, Digital Compliance. ## 4. Key Screens to Design 1. **Orchestrator Dashboard:** Executive overview and "Command Center". 2. **Global Grant Explorer:** Search agent results with advanced filtering. 3. **Analysis & Scoring Detail:** Deep dive into specific opportunities. 4. **Writing Studio:** Collaborative workspace for proposal generation. 5. **Compliance & Submission Hub:** Admin agent's checklist and document repository.
